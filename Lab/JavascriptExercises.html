4.   What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Answer:

Unobtrusive Javascript allows separation of website into 3 major categories:

Content (HTML) What is it?

Presentation (CSS) how does it look?

Behavior (Javascript) How does it respond to user interaction?

The main reason is that remove all JavaScript code from the HTML body. Its main application is to facilitate progressive enhancement and separation of functionality from content/presentation.
