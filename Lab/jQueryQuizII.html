  1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 

    2.  

              //code fragment 1  
$("li").each(function(idx, e) {  
    $(e).css(“color”, “yellow”); });  
 
//code fragment 2  
$("li").each(function() {  
    $(this).css(“color”, “yellow”); });  
 
//code fragment 3  
$("li").each(function(idx) {      $(this).css(“color”, “yellow”); });

    Answer: They are equal, and this refers to the same thing. Each member elements in li. And idx is the index, which serve no purpose in the three code fragments
