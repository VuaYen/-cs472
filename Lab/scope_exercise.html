  2. What is the difference between a method and function?

    3.  Answer: Function is a set of code to do some task in JavaScript. Method is also a function which is used as a property in Object.

  3. What does 'this' refer to when used in a Java method?

    4.  Answer: It might refer to the class itself, the methods of the class or the private instance variables of the class.

  4. What does 'this' refer to when used in a JavaScript method?

    5.  Answer: It refers to the "owner" of the method.

  5. What does 'this' refer to when used in a JavaScript constructor function?

    6.  Answer: The keyword this inside the constructor function points to the newly created object.

  6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to?

    7.  Answer: It first refers to x(own), if property is not found it might refer to y(inherited) property.

  7. What is a free variable in JavaScript?

    8.  Answer: Free variables are simply the variables that are neither locally declared nor passed as parameter.

  10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

    11.  function product() {
    let p=1;
    for(let i=0; i<arguments.length; i++) {
    p*=arguments[i];
    }

    }
