jQuery Quiz I

  1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

    2. Answer: The jQuery function refers to the global jQuery function/object or the $ function depending on the context. jQuery object is the object returned by the jQuery function that often represents a group of elements

  2. Write jQuery code to find all h1 elements that are children of a div element and make their background
    3. color red.
    Sample HTML:
    <body>
    <h1>abc</h1><br><div>
    <h1>div-1</h1>
    <h1>div-2</h1>
    </div>
    <h1>xyz</h1>
    </body>

    Answer: 

        $(document).ready(function(){
         $("div h1").css("background-color","red");
    });
